# ✍️ Air Pen - Sanal El Takibi ile Çizim Uygulaması

Bu proje, bilgisayar kamerasını kullanarak kullanıcıların işaret parmağı hareketleriyle ekrana temassız bir şekilde çizim yapmasını sağlar. 

## 🚀 Canlı Demo
Projeyi anında denemek için buraya tıklayın:  
👉 **[ahmet-air-pen.netlify.app](https://ahmet-air-pen.netlify.app)**

## ✨ Özellikler
* **Gerçek Zamanlı El Takibi:** Google MediaPipe kütüphanesi ile yüksek hassasiyetli takip.
* **Havadaki Renk Paleti:** Ekranın üst kısmındaki sanal butonlara dokunarak (temassız) renk değişimi.
* **Hafızalı Çizim Sistemi:** Her çizgi kendi rengini saklar, renk değişiminde eski çizgiler bozulmaz.
* **Gelişmiş Kontroller:**
  - `Shift` tuşuna basılı tutarak yazma modu.
  - `Ctrl` tuşuna basılı tutarak silgi modu.
  - `Space` tuşu ile tüm ekranı temizleme.
  - `S` tuşu ile yapılan çizimi `.png` formatında bilgisayara kaydetme.

## 🛠️ Kullanılan Teknolojiler
* **HTML5 & CSS3**
* **JavaScript** (Vanilla JS)
* **MediaPipe Hands API** (Yapay zeka tabanlı el takibi)

Bu proje herhangi bir kurulum gerektirmez. Yukarıdaki canlı demo linkinden veya `index.html` dosyasını bir yerel sunucu (Live Server) üzerinden açarak kullanabilirsiniz.
